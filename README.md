### Introduction

Pw is a password generator. The passwords it makes are alpanumeric strings that contain at least one digit.

### Quick Start

Compile:

```
$ make
```
or
```
$ go build pw.go
```

Run:

```
$ pw
YVSBve2ec7izj4y
```
